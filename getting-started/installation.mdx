VShop is distributed as a direct APK download for Android. It is not available on the Google Play Store, so you need to allow installation from unknown sources before installing.

## System requirements

- Android device
- Active internet connection
- Riot Games account

<Note>
iOS is not publicly distributed. There is no App Store listing for VShop at this time.
</Note>

## Install VShop

<Steps>
  <Step title="Enable installing unknown apps">
    Android blocks installations from sources outside the Play Store by default. You need to grant permission before installing the APK.

    1. Open **Settings** on your Android device.
    2. Navigate to **Apps** (or **Apps & notifications**) → **Special app access** → **Install unknown apps**.
    3. Select the browser or file manager you will use to download the APK.
    4. Toggle **Allow from this source** on.

    <Warning>
      Allowing unknown app installations reduces a security safeguard on your device. Only enable it for trusted sources. You can turn this setting off again after installing VShop.
    </Warning>
  </Step>

  <Step title="Download the APK">
    Download the latest VShop APK directly from GitHub Releases:

    ```
    https://github.com/VShopApp/mobile/releases/latest/download/VShop.apk
    ```

    Tap the link on your Android device or transfer the downloaded file to your device. The current release is **v3.1.2** (package `dev.vasc.vshop2`).

    <Tip>
      Download the APK directly on your Android device using its browser so the file is immediately accessible from the notification tray.
    </Tip>
  </Step>

  <Step title="Install the APK">
    Once the download is complete:

    1. Open the downloaded `VShop.apk` file from your notifications or file manager.
    2. Tap **Install** when prompted.
    3. Wait for the installation to finish, then tap **Open**.

    VShop will launch and walk you through the first-time setup wizard.
  </Step>
</Steps>

## Next steps

After installing, follow the [first-time setup](/getting-started/setup) guide to select your region and sign in with your Riot Games account.
